Korga Backend

This Korga server does not have an optimized production build of the frontend in its web root.

To use the frontend in development setups run npm run serve in the webapp directory.

If you get this page in production from an official Docker container, please report an issue on GitHub.
In case you are not using our Docker container, make sure that you have built the frontend with npm run build and copied the contents of webapp/dist to the wwwroot folder of the ASP.NET Core backend server.
